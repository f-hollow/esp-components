# I2C Driver Component for the T9602 Temperature and Humidity Sensor

## Features

* Get Temperature
* Get Humidity

## Get Started

To get the component, use the command `add-dependency` inside your project folder.

```bash
idf.py add-dependency "pedrominatel/t9602"
```
